Our Vision

The current cost of Invisalign and related products in the United States ranges from $3,500 to $8,000 with the national average at about $5,000. This is much more than the cost of materials to create these products. Our goal is to harness emerging technologies such as 3D printing and scanning to make the process of obtaining and using corrective retainers something everyone can afford. This will be accomplished by using graphical software to model a user’s teeth, suggest alignment corrections, and finally 3D print a plastic retainer.

Open Source

This project will be open source. The goal from start to finish will be to save the end user money.We hope the communities of graphics and 3D printing enthusiasts will contribute to create high powered tools and solutions.
